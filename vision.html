Vision

Vision is a data visualisation tool for Scania. It is available as an executable file (.exe) and as a compressed folder (.zip) containing the executable and other dependencies in a separate folder.
Note that the single file version should not be given administrative privileges as there is an unlikely but not impossible way in which a malicious attacker could corrupt one of the shared libraries in the temp folder while the bootloader is preparing it. It is therefore recommended that all environment issues are solved manually instead of automatically while using the single-file version.

The program requires SQL Express to be installed and running on the system. This is a free Microsoft software available for download at https://www.microsoft.com/sv-se/sql-server/sql-server-downloads.
